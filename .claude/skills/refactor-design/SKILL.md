# Refactor Design

Audit the codebase architecture and fix issues.

## Steps

1. **Read CLAUDE.md** for project context
2. **Identify issues** — code duplication, tight coupling, missing abstractions
3. **Plan refactoring** — what to change and why
4. **Implement changes** — one file at a time, testing after each change
5. **Run full test suite** to verify nothing is broken
6. **Update documentation** if architecture changed
